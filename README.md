# Phishing Awareness Analysis

## Objective

The objective of this project is to identify phishing emails by analyzing suspicious links, sender information, language, and social engineering techniques.

## Skills Learned

- Phishing Detection
- Threat Analysis
- Email Security
- Cybersecurity Awareness

## Tools Used

- Google Chrome
- Microsoft Word
- GitHub

## Project Outcome

Successfully identified phishing emails by examining red flags, malicious links, and deceptive tactics while distinguishing them from legitimate communications.
